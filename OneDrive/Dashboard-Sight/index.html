🎯 O que é o Sight e por que esta pesquisa existe
Sight = produto de visão computacional para varejo físico (spin-off Sekron) que coleta dados automáticos de comportamento em PDVs — fluxo, atenção, tempo em seção, cross-sell. Esta pesquisa valida se a falta de dados físicos é uma dor real do varejo e quais funcionalidades construir. Prospecção via LinkedIn com enquadramento acadêmico UnB.
