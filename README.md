# LinkedIn Boost

Projeto de prospecção no LinkedIn, desenvolvido para automatizar e otimizar o alcance de novos leads.

## Funcionalidades
- Automação de buscas e interações.
- Integração com IA para geração de mensagens personalizadas (Gemini).
- Dashboard local em FastAPI para gestão de leads.

## Como usar
1. Configure o arquivo `.env` com suas credenciais.
2. Instale as dependências:
   ```bash
   pip install -r requirements.txt
   ```
3. Inicie o servidor:
   ```bash
   python main.py
   ```

## Desenvolvimento
Este repositório é gerenciado via Git Worktrees para integração com ferramentas de assistência de IA (Claude).
